# TeamTalk NG Sound Packs

Place default TeamTalk NG sound files directly in this `Sounds` folder. Place alternate sound packs in their own subfolders.

```text
Sounds\
  connected.wav
  disconnected.wav
  userjoined.wav
  userleft.wav
  My Pack\
    connected.wav
    disconnected.wav
    userjoined.wav
    userleft.wav
```

Supported event names are matched case-insensitively and punctuation is ignored, so `userjoined.wav`, `user_joined.wav`, and `User Joined.wav` are equivalent.

Recommended event file names:

- `connecting.wav`
- `connected.wav`
- `disconnected.wav`
- `joinedchannel.wav`
- `channelmessage.wav`
- `directmessage.wav`
- `userjoined.wav`
- `userleft.wav`
- `filetransferstarted.wav`
- `filetransferfinished.wav`
- `filetransferfailed.wav`
- `filetransfercanceled.wav`
- `pushtotalkenabled.wav`
- `pushtotalkdisabled.wav`
- `voiceactivationenabled.wav`
- `voiceactivationdisabled.wav`
- `videostarted.wav`
- `videostopped.wav`
- `desktopsharestarted.wav`
- `desktopsharestopped.wav`
